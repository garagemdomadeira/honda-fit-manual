Descrição do Sistema do A/C

O sistema do ar condicionado retira o calor do compartimento do passageiro, fazendo circular o refrigerante através do sistema, conforme em baixo ilustrado.

 

Este veículo utiliza refrigerante HFC-134a (R-134a) que não contém clorofluorcarbonatos. Tenha em atenção os seguintes itens de manutenção:

  • Não misture os refrigerantes CFC-12 (R-12) e HFC-134a (R-134a). Estes não são compatíveis.
  • Utilize somente o óleo refrigerante de polialcalenoglicol (PAG) recomendado (KEIHIN, SP-10) concebido para o compressor R-134a. A mistura do óleo refrigerante recomendado (PAG) com qualquer outro óleo refrigerante resultará numa avaria do compressor.
  • Todos os componentes do sistema do A/C (compressor, linha de descarga, linha de aspiração, evaporador, condensador, depósito desidratador, válvula de expansão, O-rings para as uniões) terão que ser apropriados para o refrigerante R-134a. Não confunda com as peças para o R-12.
  • Utilize um detector de fugas de gás halogéneo concebido para o refrigerante R-134a.
  • Os equipamentos de manutenção de refrigerante R-12 e R-134a não são intermutáveis. Ao efectuar a manutenção do sistema do ar condicionado R-134a, utilize exclusivamente um sistema de recuperação/reciclagem/carga para refrigerante R-134a.
  • Proceda sempre à recuperação de refrigerante R-134a com um sistema aprovado de recuperação/reciclagem/carga, antes de desligar qualquer componente do sistema de A/C.
