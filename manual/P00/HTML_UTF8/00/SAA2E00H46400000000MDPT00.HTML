Substituição/Inspecção dos Componentes Após a Insuflação do SRS

NOTA: Antes de proceder a quaisquer reparações no sistema SRS, use o método do menu do SRS PGM Tester para verificar se existem DTC; consulte o Índice do Diagnóstico de Avarias por DTC quanto aos componentes que são activados de forma menos evidente (tensores dos cintos de segurança)

Após uma colisão em que os airbag(s) dianteiro(s)se tenha(m) insuflado, substitua os seguintes itens:
  • Unidade SRS
  • Airbag(s) insuflado(s)
  • Tensores dos cintos de segurança
  • Sensores dianteiros

Após uma colisão em que os airbag(s) lateral(ais) se tenha(m) insuflado, substitua os seguintes itens:
  • Unidade SRS
  • Airbag(s) lateral(ais) insuflado(s)
  • Sensor(es) de impacto lateral para o(s) lado(s) activado(s)

Durante o processo de reparação, inspeccione as seguintes zonas:
  • Inspeccione as cablagens do SRS. Substitua, não tente reparar, quaisquer cablagens danificadas.
  • Verifique se não há avarias nos enroladores de cabo provocadas pelo calor. Se houver quaisquer danos, substitua os enroladores de cabo.

Depois de o veículo ter sido completamente reparado, ligue a ignição. Se o indicador SRS se acender por cerca de 6 segundos e, em seguida, se apagar, o sistema de airbags do SRS está bom. Se o indicador não funcionar adequadamente, utilize o Método do Menu SRS do PGM Tester para ler o(s) DTC(s). Se este não for capaz de recolher quaisquer códigos, utilize o método do menu SCS do Ensaiador Se o método SCS também não funcionar, poderá ter de instalar uma unidade SRS que se saiba estar em perfeitas condições, para ler o(s) DTC(s). Se mesmo assim não conseguir ler nenhum código, consulte o Diagnóstico de Avarias do Circuito do Indicador SRS.
